# 文档目录

本目录收拢项目内仍有维护价值的设计、现状和实现文档。

## 结构

- `status/`：前后端现状快照，适合快速了解历史状态，但执行前应以当前代码为准。
- `plans/`：产品、页面和中长期演进方案。
- `architecture/`：后端能力设计与关键实现说明。
- `frontend/`：前端脚手架、组件和局部实现说明。

## 当前保留范围

- 已将原本散落在仓库根目录和源码目录中的高价值 Markdown 迁入此处归档。
- `README.md`、`AGENTS.md`、`openspec/` 下文档仍保留在原位，因为它们分别承担仓库入口、代理规则和规范管理职责。
