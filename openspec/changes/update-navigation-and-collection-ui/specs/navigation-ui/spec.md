## MODIFIED Requirements

### Requirement: 底部导航入口数量控制
系统 MUST 将底部一级导航入口数量控制在 4~5 个以内，避免移动端点击区域过小。

#### Scenario: 底部导航入口精简
- **WHEN** 用户进入任意主页面
- **THEN** 底部 Tab 仅展示 4 个主入口（首页/衣橱/搭配中心/我的），其余搭配相关入口下沉到二级

### Requirement: 搭配中心二级导航
系统 MUST 在“搭配中心”提供二级导航，支持在搭配预览、场景推荐、搭配合集之间快速切换。

#### Scenario: 切换到场景推荐
- **WHEN** 用户在搭配中心选择“场景推荐”
- **THEN** 展示场景推荐页面内容且底部 Tab 仍高亮“搭配中心”

#### Scenario: 切换到搭配合集
- **WHEN** 用户在搭配中心选择“搭配合集”
- **THEN** 展示套装库列表页面内容且支持进入创建页
